# Projet Base de Données Université

## Description
Ce projet consiste à concevoir et implémenter une base de données relationnelle permettant de gérer les informations d’un établissement d’enseignement supérieur.

Le système prend en charge :
- les étudiants
- les enseignants
- les cours
- les inscriptions

---

## Objectifs
- Modéliser un système académique (MCD, MLD, MPD)
- Implémenter la base de données avec SQL Server
- Manipuler les données avec des requêtes SQL
- Tester et valider le fonctionnement du système
